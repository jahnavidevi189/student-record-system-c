Student Record System in C

📌 Description

This is a simple C program that allows users to input and display student details such as ID, name, and marks.

🚀 Features

- Store details of 3 students
- Input student data from user
- Display formatted student records

🛠️ Technologies Used

- C Programming Language

▶️ How to Run

1. Clone the repository:
   git clone https://github.com/your-username/student-record-system-c.git
2. Navigate to the folder:
   cd student-record-system-c
3. Compile the program:
   gcc main.c -o student
4. Run the program:
   ./student
